# PhaST
Customized and patched st config written by me for me!
1. [Installation](#installation)
2. [Patches](#patches)

## Installation
To install you have to clone the repo and make install.
```bash
git clone https://github.com/Phandal/PHAST
cd PHAST
sudo make -B clean install
```
